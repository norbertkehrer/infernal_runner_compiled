Infernal Runner Compiled to JavaScript

This is a JavaScript port of the Amstrad CPC game "Infernal Runner" made by Eric Chahi in 1985. It is built from the virtual machine interpreter code of Gregory Montoir. Norbert Kehrer compiled the bytecode to native JavaScript in order to eliminate the intrepretive layer and having the whole game as a pure JavaScript program.

Use the arrow keys to control the player and Space or Enter to jump.
